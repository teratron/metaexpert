# Product Guide

## Target Users

MetaExpert is primarily designed for developers who are building trading bots and algorithmic trading strategies for the cryptocurrency markets.

## Goals

The main goals of MetaExpert are to:

*   Provide a unified API for interacting with multiple cryptocurrency exchanges, abstracting away the complexities of each platform.
*   Offer a high-performance backtesting engine for strategy validation.
*   Simplify the development of complex trading algorithms by providing a clear and extensible framework.

## Key Features

To achieve its goals, MetaExpert provides the following key features:

*   **Multi-Exchange Support:** A unified interface for major cryptocurrency exchanges.
*   **Structured Logging:** An advanced logging system with context management and automatic filtering of sensitive data.
*   **Comprehensive Backtesting:** In-depth backtesting capabilities for strategy validation.
*   **Modular Architecture:** A clean, extensible codebase that follows SOLID principles, allowing for easy customization and extension.
