# Tech Stack

## Programming Language

*   **Python:** The primary programming language for the project, specifically targeting Python 3.12+.

## Core Libraries

*   **Pydantic:** Used for data validation and settings management, ensuring robust and type-safe data handling.
*   **Websockets:** Utilized for real-time communication with cryptocurrency exchanges, enabling efficient data streaming.

## Development and Testing Tools

*   **Pytest:** The chosen framework for writing and running tests, supporting comprehensive unit and integration testing.
*   **Ruff:** Employed for linting and formatting the codebase, ensuring adherence to code style guidelines and maintaining code quality.
*   **Uv:** Used for efficient dependency management, ensuring consistent and reproducible build environments.
*   **MkDocs-Material:** The framework for generating comprehensive project documentation, providing a clean and user-friendly interface.

## Key Integrations

*   **Cryptocurrency Exchanges:** The system integrates with several major cryptocurrency exchanges, including Binance, Bybit, OKX, and Mexc, providing a unified interface for various trading operations.
